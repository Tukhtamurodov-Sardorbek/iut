## Why Multi-Module?
- Faster build times (incremental builds)
- Clearer separation of concerns
- Reusability of modules
- Team scalability

## Project Structure
- `:app` - The main entry point
- `:core` - Shared logic and utilities
- `:feature-X` - Specific feature modules
- `:data` - Repository and networking layers
